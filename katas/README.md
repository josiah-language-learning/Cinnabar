# Katas

A kata is a focused repetition on a single concept. The point is not to solve a problem once and move on — it is to build muscle memory by doing the same thing cleanly enough that the pattern becomes available without effort.

Each kata here has a specific Mercury or logic programming concept it is drilling. Work the concept, not just the solution. If you finish a kata and couldn't explain which mode or determinism annotation you chose and why, you haven't finished it.
